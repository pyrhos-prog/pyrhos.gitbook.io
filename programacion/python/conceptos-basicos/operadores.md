# Operadores

Los operadores fundamentales son:&#x20;

* Suma (+): suma numeros y secuencias como cadenas y listas
* Resta (-):  se utiliza para restar un número de otro
* Multiplicación (\*): se multiplica un número por otro o se repite una cadena o lista el numero de veces especificada
* División (/): Divide un número entre otro.
* Exponente (\*\*): Eleva un número a la potencia de otro.
