# Introducción a Python

### Que es Python?

Python es un lenguaje de programación de alto nivel, utiliza una sintaxis que favorece un codigo legible.

### Historia de Python

Python fué creado por Guido van Rossum y lo lanzo en 1991. Guido van Rossum creó python con la idea de mejorar la productividad de los programadores y la legibilidad del código.

Por su filosofia, además de ser mas fácil de aprender que otros lenguajes como C o Java también permite crear programas complejos con menos lineas de código que otros lenguajes.

Uno de los documentos mas importante en la comunidad de Python es [El zen de Python](https://elpythonista.com/zen-de-python), este documento contiene la esencia de la filosofia de Python.
