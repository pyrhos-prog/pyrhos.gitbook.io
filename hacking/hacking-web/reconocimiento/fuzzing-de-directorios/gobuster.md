# gobuster

es otro directorio web y fuzzer de archivos popular, es conocido por su velocidad y simplicidad.

Puedes instalarlo `GoBuster` usando el siguiente comando:

```
 go install github.com/OJ/gobuster/v3@latest
```

#### Casos de uso <a href="#use-cases-1" id="use-cases-1"></a>

| `Content Discovery`           | Escanee y encuentre rápidamente contenido web oculto, como directorios, archivos y hosts virtuales. |
| ----------------------------- | --------------------------------------------------------------------------------------------------- |
| `DNS Subdomain Enumeration`   | Identificar subdominios de un dominio de destino.                                                   |
| `WordPress Content Detection` | Utilice listas de palabras específicas para encontrar contenido relacionado con WordPress.          |
