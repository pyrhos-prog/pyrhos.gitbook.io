# Enumeración inicial

| Comando                                                                                             | Descripción                                                                                                                                                                                                                                                                                                                                                                    |
| --------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `nslookup ns1.inlanefreight.com`                                                                    | Se utiliza para consultar el sistema de nombres de dominio y descubrir la asignación de la dirección IP al nombre de dominio del destino introducido desde un host basado en Linux.                                                                                                                                                                                            |
| `sudo tcpdump -i ens224`                                                                            | Se utiliza para comenzar a capturar paquetes de red en la interfaz de red siguiendo la opción de un host basado en Linux. `-i`                                                                                                                                                                                                                                                 |
| `sudo responder -I ens224 -A`                                                                       | Se utiliza para comenzar a responder y analizar consultas en la interfaz especificada procediendo a la opción y operando en modo que se activa usando LLMNR, NBT-NS y mDNS. Se realiza desde un host basado en Linux. Opciones mencionadas: `-I` (interfaz) y `-A` (Passive Analysis).                                                                                         |
| `fping -asgq 172.16.5.0/23`                                                                         | Realiza un barrido de ping en el segmento de red especificado desde un host basado en Linux.                                                                                                                                                                                                                                                                                   |
| `sudo nmap -v -A -iL hosts.txt -oN /home/User/Documents/host-enum`                                  | Realiza un análisis de nmap con la detección del sistema operativo, la detección de versiones, el análisis de scripts y traceroute habilitados (`-A`) en función de una lista de hosts (`hosts.txt`) especificados mediante la opción `-iL`. A continuación, envía los resultados del análisis al archivo especificado con `-oN`. Se realiza desde un host basado en Linux.    |
| `sudo git clone https://github.com/ropnop/kerbrute.git`                                             | Se utiliza para clonar la herramienta kerbrute desde un host basado en Linux.                                                                                                                                                                                                                                                                                                  |
| `make help`                                                                                         | Se utiliza para enumerar las opciones de compilación que son posibles desde un host basado en Linux (`make`).                                                                                                                                                                                                                                                                  |
| `sudo make all`                                                                                     | Se utiliza para compilar un binario para múltiples plataformas de sistemas operativos y arquitecturas de CPU (Kerbrute).                                                                                                                                                                                                                                                       |
| `./kerbrute_linux_amd64`                                                                            | Se utiliza para probar el binario compilado elegido desde un host basado en Linux (Kerbrute).                                                                                                                                                                                                                                                                                  |
| `sudo mv kerbrute_linux_amd64 /usr/local/bin/kerbrute`                                              | Se utiliza para mover el binario a un directorio que se puede configurar para que esté en la ruta de un usuario de Linux, facilitando el uso de la herramienta (Kerbrute).                                                                                                                                                                                                     |
| `./kerbrute_linux_amd64 userenum -d INLANEFREIGHT.LOCAL --dc 172.16.5.5 jsmith.txt -o kerb-results` | Ejecuta la herramienta Kerbrute para detectar nombres de usuario en el dominio (`INLANEFREIGHT.LOCAL`) especificado mediante la opción `-d` y el controlador de dominio asociado especificado mediante `--dc`, usando una lista de palabras (`jsmith.txt`) y envía los resultados al archivo especificado con `-o` (`kerb-results`). Se realiza desde un host basado en Linux. |
