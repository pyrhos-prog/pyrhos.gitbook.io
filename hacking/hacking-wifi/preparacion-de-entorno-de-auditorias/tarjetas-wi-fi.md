# Tarjetas Wi-Fi

La antena Wi-Fi es de los componentes mas importantes para una auditoria, no todas las antenas sirven para auditar, se necesita poder capturar señales débiles, activas que tenga modo monitor y poder inyectar paquetes.

### Componentes clave de una antena Wi-Fi

* **Chipset**: determina la compatibilidad con modo monitor
* **Antena**: Recibe y transmite la señal
* **Conectores**: permite montar diferentes antenas

#### Características para elegir la antena

* **Compatibilidad de frecuencias:** Hay que comprobar si la antena es compatible con las bandas frecuencias que queramos auditar, no todas sirven para 5GHz.
* **Compatibilidad:** Hay que comprobar que la antena Wi-Fi es compatible con el Sistema Operativo que vayamos a usar (Windows, Linux, Mac).
* **Soporte para modo monitor e inyección:** Si la antena no tiene estas 2 opciones no vamos a poder hacer una auditoria con ella.
