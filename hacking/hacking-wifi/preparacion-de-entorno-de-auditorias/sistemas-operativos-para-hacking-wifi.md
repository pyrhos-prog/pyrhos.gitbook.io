# Sistemas operativos para hacking wifi

Podemos auditar redes Wi-Fi con prácticamente cualquier sistema operativo, pero algunos sistemas operativos vienen ya configurados con las herramientas necesarias para ello, facilitándonos el trabajo.



### Sistemas Operativos recomendados

{% tabs %}
{% tab title="Kali Linux" %}
#### Características

* Esta basado en Debian
* Tiene una gran cantidad de herramientas presintaladas
* Soporta muchos drivers Wi-Fi

#### Herramientas para hacking Wi-Fi

* aircrack-ng
* airmon-ng&#x20;
* aireplay-ng
* hcxdumptool
* Wireshark
{% endtab %}

{% tab title="Parrot OS" %}
#### Características

* Es mas ligero que Kali Linux
* Esta basado en Debian
* Funciona mejor en equipos de pocos recursos

#### Uso recomendado

* Auditorias ligeras
* Ordenadores con pocos recursos
{% endtab %}

{% tab title="BlackArch" %}
#### Características

* Esta basado en Arch Linux
* Requiere mas conocimiento técnico de linux
* Tiene un gran repositorio de herramientas de hacking

#### Uso recomendado

* Usuarios experimentados&#x20;
{% endtab %}

{% tab title="Wifislax" %}
### Características

* Sistema Operativo enfocado en auditorias Wi-Fi
* Esta basado en Slackware
* Tiene scripts automatizados

#### Uso recomendado

* Auditorias Wi-FI concretas
{% endtab %}
{% endtabs %}
