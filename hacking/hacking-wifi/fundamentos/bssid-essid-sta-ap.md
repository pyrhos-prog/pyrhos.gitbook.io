# BSSID, ESSID, STA, AP

### AP (Access Point)

Un Access Point es un dispositivo que ofrece red Wi-Fi.

**Funciones Principales:**

* Emite la señal inalámbrica.
* Gestiona las conexiones.
* Intermedia entre clientes y la red cableada.

**Ejemplos:**

* Routers
* Puntos de acceso empresariales
* Repetidor configurado como AP

### STA (Station)

Una station es cualquier dispositivo que se conecte a un AP.

* Portátiles
* Móviles
* Dispositivos IoT

### ESSID (Extended Service Set Identifier)

El ESSID es el nombre visible de la red Wi-Fi.

```
Wifi_casa
```

* Es el nombre que ve el usuario
* Pueden tenerlo varios APs
* No identifica a un AP en concreto

### BSSID (Basic Service Set identifier)

EL BSSID identifica de manera única a un AP.

```
00:11:22:33:44:55
```

* Es una dirección mac
* Cada AP tiene su propio BSSID aunque tengan el mismo ESSID
