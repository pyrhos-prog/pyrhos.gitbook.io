# Bandas 2.4 GHz y 5 GHz

Las bandas Wi‑Fi definen en qué rango de frecuencias se transmite la señal inalámbrica.\
La frecuencia influye directamente en alcance, velocidad, interferencias y estabilidad.

## Banda 2.4 GHz&#x20;

Esta banda es la mas antigua y extendida a nivel mundial.

### Características

|                            |                                                                                                                                                                                  |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Alcance y propagación**  | <ul><li>Tiene más alcance</li><li>Atraviesa mejor los obstáculos</li><li>Se degrada menos que la 5G con la distancia</li></ul>                                                   |
| **Velocidad**              | <ul><li>Velocidades mas bajas</li><li>Usa canales estrechos</li><li>Tiene menos capacidad para manejar muchos dispositivos</li></ul>                                             |
| **Canales y solapamiento** | <ul><li>Los canales están muy juntos entre sí</li><li>La mayoría de canales se solapan</li><li>Solo los canales 1, 6 y 11 no se pisan entre ellos</li></ul>                      |
| **Interferencias**         | <p>La banda 2.4GHz no es solo de Wi-Fi, en ella tambien hay:</p><p></p><ul><li>Bluetooth</li><li>Microondas</li><li>Dispositivos domóticos</li><li>Mandos inalámbricos</li></ul> |

## Banda 5 GHz&#x20;

Esta banda es la mas moderna, con mejoras de rendimiento y estabilidad

### Características

|                            |                                                                                                                                                       |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Alcance y propagación**  | <ul><li>Tiene menos alcance</li><li>Atraviesa peor los obstáculos</li><li>Se degrada más con la distancia</li></ul>                                   |
| **Velocidad**              | <ul><li>Velocidades mucho mayores</li><li>Soporta canales mas anchos</li><li>Puede manejar muchos mas dispositivos a la vez</li></ul>                 |
| **Canales y solapamiento** | <ul><li>Tiene muchos canales más disponibles</li><li>Menos solapamiento</li></ul>                                                                     |
| **Interferencias**         | <ul><li>Esta menos congestionado </li><li>Tienes menos dispositivos que no son de Wi-Fi</li><li>Tiene una señal mas limpia que la de 2.4GHz</li></ul> |
