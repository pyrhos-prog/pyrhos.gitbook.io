# Metasploit

## Introducción

Metasploit es una framework de explotación de vulnerabilidades que contiene módulos, herramientas y técnicas que permiten:

* Analizar vulnerabilidades
* Probar configuraciones inseguras
* Ejecutar exploits de forma controlada
* Validar si un sistema es vulnerable
* Realizar una post explotación con permisos adecuados
* Crear módulos de pruebas personalizadas
