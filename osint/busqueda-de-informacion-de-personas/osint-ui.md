# Osint UI

**Osint-UI es una pagina web  creada por** [**@afsh4ck**](https://www.youtube.com/@afsh4ck) **con opciones muy pontentes para un análisis OSINT como:**

* búsqueda de usuarios
* emails
* números de teléfono
* información de dominios
* escaneo de puertos
* comprobación de reputación de un dominio
* análisis de metadatos de imágenes
* analisis de hashes
