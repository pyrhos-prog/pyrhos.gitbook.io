# PimEyes

### PimEyes: El Motor de Búsqueda Facial Inverso para la Web Abierta

PimEyes es una herramienta de búsqueda facial inversa que permite a los usuarios encontrar imágenes de personas en la web utilizando una fotografía como punto de partida. A diferencia de los motores de búsqueda de imágenes tradicionales que buscan objetos o escenas, PimEyes se especializa en la detección y coincidencia de rostros humanos.

### ¿Cómo funciona?

**El proceso es sencillo y directo:**

1. Carga de la Imagen: El usuario sube una o varias fotografías de un rostro. Cuantas más imágenes se proporcionen y más claras sean, mayor será la precisión de los resultados.
2. Análisis Biométrico: PimEyes utiliza algoritmos avanzados de reconocimiento facial para analizar las características biométricas únicas del rostro en la imagen cargada.
3. Búsqueda en la Base de Datos: Este perfil facial se compara con una vasta base de datos de imágenes indexadas de la "web abierta" (sitios web públicos, blogs, noticias, foros, etc.). Es importante destacar que PimEyes no busca en redes sociales privadas o contenido protegido por contraseñas.
4. Resultados y Dominios: La herramienta devuelve un conjunto de imágenes que coinciden o son muy similares al rostro original, junto con los dominios web donde fueron encontradas. Como señalaste, puede haber falsos positivos o imágenes que no coinciden perfectamente, especialmente si la calidad de la foto original es baja o hay personas con rasgos muy similares.
