# Kali Linux

## ¿Que es Kali Linux?

Kali linux es una distribución basada en [Debian](debian.md) hecha específicamente para **seguridad informática**, **auditorías**, **análisis forense digital** y **pruebas de penetración (pentesting).** La desarrolló Offensive Security.

<div data-with-frame="true"><figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure></div>

Kali Linux trae herramientas preinstaladas para diferentes tareas:

* Análisis de vulnerabilidades
* Pruebas de penetración&#x20;
* Ingenieria inversa
* Recopilación de información&#x20;
* Explotación de sistemas
* Análisis forense
* Pruebas de seguridad inalámbrica
