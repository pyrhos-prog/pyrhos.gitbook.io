# Estructura de directorios del sistema

Esta es la estructura de directorios de un sistema linux:

```
/
├── bin        → Binarios esenciales
├── boot       → Archivos de arranque (kernel, GRUB)
├── dev        → Dispositivos del sistema
├── etc        → Archivos de configuración
├── home       → Carpetas de usuarios
├── lib        → Bibliotecas esenciales
├── mnt        → Montajes temporales
├── media      → Medios extraíbles (USB, CD, etc.)
├── opt        → Software adicional
├── proc       → Información del sistema (virtual)
├── root       → Carpeta personal de root
├── sbin       → Binarios del administrador
├── srv        → Datos de servicios/servidores
├── tmp        → Archivos temporales
├── usr        → Programas instalados y recursos
├── var        → Datos variables (logs, spool…)
└── sys        → Información del kernel

```
