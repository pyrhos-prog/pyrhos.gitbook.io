# Comando id

El comando `id`  se utiliza para mostrar a que grupos pertenece nuestro usuario y el identificador de cada uno de ellos.

```
pyrhos@debian:~$ id
uid=1000(pyrhos) gid=1000(pyrhos) grupos=1000(pyrhos),24(cdrom),25(floppy),29(audio),30(dip),44(video),46(plugdev),100(users),101(netdev),102(scanner),106(bluetooth),108(lpadmin),117(libvirt),993(kvm)

```
