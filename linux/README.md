# Introducción a Linux

### ¿Que es Linux?

Linux es un kernel Open Source basado en Unix creado por Linus Torvald, a diferencia de Windows o MacOS, linux es de codigo abierto por lo que cualquiera puede modificar su codigo fuente y crear sus propias versiones de Linux.

#### Filosofía del software libre

* **Transparencia:** cualquier persona puede leer y modificar el código.
* **Colaboración:** miles de desarrolladores contribuyen en comunidad.
* **Libertad:** los usuarios pueden adaptar el sistema según sus necesidades.

### Ventajas de Linux

* Es Open Source por lo tanto cualquier persona puede ver y modificar el código.
* Su comunidad es muy activa, crean distribuciones, dan soporte y ayudan a los nuevos de manera gratuita.
* No necesita muchos recursos para funcionar.
* Es compatible con una amplia gama de hardware, desde componentes muy viejos a componentes muy nuevos.
